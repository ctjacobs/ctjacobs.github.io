Software

Below is a partial list of free, open-source software projects that I develop. ## Computational fluid dynamics

  • [Fluidity](http://www.fluidity-project.org/): A computational fluid dynamics code with adaptive unstructured mesh capabilities.
  • [Firedrake-Fluids](https://github.com/firedrakeproject/firedrake-fluids): A computational fluid dynamics code which uses the [Firedrake](http://www.firedrakeproject.org/) automated solution framework.

## Amateur radio * [PyQSO](http://christianjacobs.uk/pyqso): A contact logging tool for amateur radio operators. ## Research data management * [PyRDM](http://github.com/pyrdm/pyrdm): A Python-based library for research data management (RDM). ## Miscellaneous * [Game of Life](http://github.com/ctjacobs/game-of-life/): An implementation of Conway's Game of Life, for demonstration purposes in a Python programming workshop. * [Imperial SCR Menu](http://github.com/ctjacobs/imperialscrmenu): An Android application for checking the daily menu at Imperial's SCR restaurant.
